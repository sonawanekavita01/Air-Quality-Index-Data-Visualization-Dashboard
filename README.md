✨ Features

🌐 Interactive Visualizations: AQI, PM2.5, PM10, NO2, SO2, CO, O3 levels

🗺️ Geographical Mapping: Pollution hotspots visualized on maps

⏱️ Time-Series Analysis: Track daily, weekly, or monthly air quality trends

🔎 Custom Filters: Filter by location 📍, pollutant ⚗️, or date 📅

📊 Real-Time Insights: Monitor current air quality conditions

🛠️ Technologies Used
🛠️ Technology	🔍 Purpose
🐍 Python	Data processing & analysis
🌐 Streamlit	Interactive dashboard interface
📊 Pandas & NumPy	Data manipulation
📈 Matplotlib & Plotly	Interactive visualizations
🗺️ Folium / Geopandas	Geospatial mapping

⚡ Installation & Usage

Clone the repository

git clone https://github.com/yourusername/Air-Quality-Index-Data-Visualization-Dashboard.git
cd Air-Quality-Index-Data-Visualization-Dashboard


Install dependencies

pip install -r requirements.txt


Run the dashboard

streamlit run app.py


Open in browser – Streamlit opens the dashboard at http://localhost:8501
